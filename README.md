# Assignment 2 – AI-Powered SEO Content Production Research

## Project Overview

For the second stage of the assessment, I researched how experienced SEO professionals approach AI-powered SEO content production.

Rather than relying on a single source, I studied multiple experts across LinkedIn and YouTube to identify recurring ideas, workflows, and best practices used in modern SEO.

## Research Topic

**AI-Powered SEO Content Production**

## Repository Structure

```text
research/
├── sources.md
├── linkedin-posts/
│   └── linkedin-summary.md
├── youtube-transcripts/
│   └── youtube-summary.md
└── other/
    └── observations.md
```

## Research Process

For this project I:

- Selected ten experienced SEO professionals with strong credibility and active educational content.
- Researched LinkedIn content from Rand Fishkin, Neil Patel, Aleyda Solis, Lily Ray, Kevin Indig, Marie Haynes, and Goutam Singh.
- Researched YouTube content from Rand Fishkin, Brian Dean, Neil Patel, Matt Diggity, and Gael Breton.
- Manually collected each LinkedIn expert's recent posts related to AI search, SEO, and content strategy, recording post dates, links, and summaries written in my own words.
- Identified relevant YouTube videos for each expert and collected one transcript per creator using YouTube's built-in transcript feature.
- Initially attempted to retrieve transcripts using the Supadata API through a no-code workflow. After encountering configuration issues under time constraints, I switched to YouTube's native transcript feature to ensure reliable data collection.
- Compared recurring themes across both platforms and organized the findings into structured research notes.
- Wrote all summaries in my own words rather than copying source material, while documenting my own observations and conclusions.

## Personal Reflection

Before starting this project, I assumed AI-powered SEO was primarily about using AI to generate content faster.

After researching multiple SEO professionals, I realized the bigger shift is happening in how AI search engines discover and cite information. Several experts pointed to studies showing that AI-generated answers often rely on a wide variety of trusted third-party sources, meaning traditional Google rankings alone are no longer enough.

This completely changed how I think about AI-powered SEO. Instead of focusing only on producing more content, successful marketers now need to understand search intent, build authority across multiple platforms, and create content that is genuinely useful for both users and AI systems.

I also learned an important lesson about approaching technical work under deadlines. I initially spent significant time trying to make the Supadata API work before realizing that YouTube's built-in transcript feature would allow me to complete the research more efficiently. Going forward, I would start with the simplest reliable solution first and only introduce additional technical complexity when it provides a clear advantage.

## Repository Purpose

This repository documents my learning process and research findings throughout the 100Hires assessment. Beyond completing the assignment, my goal was to build a structured research resource that can later be expanded into a practical SEO playbook.
